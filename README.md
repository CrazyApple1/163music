# 163music

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
#### 普通模式开发环境运行:
```
npm run serve
```

#### ssr模式开发环境运行:
- 配置 host -> 163music.jacksonx.cn:8080 局域网ip:8080
- 打包编译 npm run build:all
- 运行 npm run dev
- 浏览器打开 163music.jacksonx.cn:8080

### Compiles and minifies for production
```
npm run build:all
```

### Lints and fixes files
```
npm run lint
```
